# 🔨 打地鼠游戏 🐹

一个有趣的网页打地鼠游戏，使用纯HTML、CSS和JavaScript开发。

## 🎮 游戏特色

- **响应式设计**: 适配桌面和移动设备
- **三种难度**: 简单、中等、困难
- **计分系统**: 实时显示得分和最高分
- **音效反馈**: 打击和游戏结束音效
- **暂停功能**: 可随时暂停/继续游戏
- **本地存储**: 自动保存最高分记录
- **键盘支持**: 支持空格键和R键快捷操作

## 🎯 游戏规则

1. 点击"开始游戏"按钮开始
2. 地鼠会随机从洞里冒出来
3. 快速点击地鼠获得分数（每只10分）
4. 游戏时长30秒
5. 时间结束后显示最终得分

## 🕹️ 操作说明

### 鼠标操作
- 点击地鼠进行打击
- 点击按钮控制游戏

### 键盘快捷键
- **空格键**: 开始游戏/暂停继续
- **R键**: 重置游戏

### 难度设置
- **简单**: 地鼠出现慢，停留时间长
- **中等**: 地鼠出现适中，停留时间适中
- **困难**: 地鼠出现快，停留时间短

## 🔧 自定义设置

可以通过修改 `script.js` 中的 `difficultySettings` 对象来调整游戏难度：

```javascript
this.difficultySettings = {
    easy: { minTime: 1000, maxTime: 2000, moleUpTime: 1500 },
    medium: { minTime: 600, maxTime: 1200, moleUpTime: 1000 },
    hard: { minTime: 400, maxTime: 800, moleUpTime: 700 }
};
```

- `minTime`: 地鼠出现的最小间隔时间（毫秒）
- `maxTime`: 地鼠出现的最大间隔时间（毫秒）
- `moleUpTime`: 地鼠停留时间（毫秒）
